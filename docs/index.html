What it is

stryke-scylla is a ScyllaDB / Apache Cassandra client for stryke. It speaks the native CQL binary protocol via ScyllaDB's official pure-Rust driver and ships as a cdylib dlopened in-process on the first use Scylla. The driver is async; the cdylib owns one embedded tokio runtime and blocks on each call, so the stryke-facing API is synchronous. A Session is cached per (nodes, auth, keyspace).

Install

s add github.com/MenkeTechnologies/stryke-scylla

Quick start

use Scylla

var %conn
$conn{node}     = "127.0.0.1:9042"
$conn{keyspace} = "app"

Scylla::create_keyspace("app", %conn)
Scylla::create_table("app.users", columns => "id uuid, name text", primary_key => "id", %conn)
Scylla::execute("INSERT INTO app.users (id, name) VALUES (uuid(), 'ada')", %conn)

p Scylla::count("app.users", %conn)
val @rows = Scylla::query("SELECT name FROM app.users", %conn)
